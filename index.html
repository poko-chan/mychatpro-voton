電話番号（SMS）認証

🔧 Firebase接続設定・トラブルシューティングガイド ▼
  1. Authentication の有効化
    Firebase Console ＞ Authentication ＞ Sign-in method にて、「メール/パスワード」「Google」「Apple」「Microsoft」「電話」を有効にしてください。
  2. ドメイン承認の追加
    Google/Apple/Microsoft等でログインする際、Vercelのドメイン（例: xxx.vercel.app）を Firebase Console ＞ Authentication ＞ Settings ＞ Authorized domains に追加する必要があります。
  3. Apple / Microsoft ログインのプロバイダ設定
    各プロバイダの開発者サイトで作成したクライアントIDとシークレットを、Firebase Console側に登録する必要があります。
  4. 電話番号認証（SMS）の設定
    SMSの送信にはFirebaseによる課金・クォータが適用される場合があります。テスト時はFirebaseコンソールでテスト用の電話番号とコード（例: +819012345678, 123456）を事前に登録しておくと、SMS送信なしでログインテストが可能です。
